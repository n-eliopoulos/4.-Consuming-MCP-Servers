Handling Responses in Agent Loops

MCP tool results feed back into the LLM conversation. This is the core integration pattern: the LLM decides to call a tool, the MCP client executes it, and the result goes back to the LLM as a tool result message.

The pattern

  1. User sends a message to the LLM
  2. LLM responds with a tool_use block (tool name + arguments)
  3. Agent extracts the tool call and invokes it via MCPClient
  4. Tool result is sent back to the LLM as a tool_result message
  5. LLM generates a final response incorporating the tool result
from ai_shim import ChatClient, MCPClient

chat = ChatClient(model="claude-sonnet")
mcp = MCPClient(server="market-data")
mcp.initialize()

messages = [{"role": "user", "content": "What's the current price of AAPL?"}]
response = chat.send(messages=messages, tools=mcp.list_tools())

if response.tool_calls:
    tool_call = response.tool_calls[0]
    result = mcp.call_tool(tool_call.name, tool_call.arguments)
    messages.append(response.to_message())
    messages.append({"role": "tool", "tool_use_id": tool_call.id, "content": result.text()})
    final = chat.send(messages=messages)
    print(final.text)
