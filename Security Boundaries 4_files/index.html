Logging considerations

Every hop is a potential logging point. In financial services, you must log enough for audit trails but never log PII, credentials, or sensitive financial data. Log the tool name and sanitized arguments — not the raw content.

Every hop in the MCP data chain is a potential data leak. For each integration, trace the full chain and identify what sensitive data is visible at each point.
