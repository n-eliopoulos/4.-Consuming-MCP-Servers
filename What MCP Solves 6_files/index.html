Consuming MCP Servers

Module 1: What MCP Solves

The MCP Ecosystem

MCP's value grows with its ecosystem. The more servers available, the more capabilities any MCP client can access without custom integration work.

Community servers

Anthropic maintains a registry of community-built MCP servers for common integrations:

  • Filesystem — read/write files, list directories
  • GitHub — create issues, search repos, manage PRs
  • Slack — send messages, search channels, manage threads
  • Databases — PostgreSQL, SQLite, DynamoDB query and schema inspection
  • Web search — Brave Search, Google Search integration

Enterprise patterns

Organizations build internal MCP servers that wrap proprietary systems behind the standard protocol:

  • Internal APIs — wrap REST/GraphQL services as MCP tools
  • Compliance tools — rule lookups, trade surveillance, regulatory checks
  • Market data feeds — real-time quotes, historical prices, index data
  • Analytics engines — query interfaces, report generation, dataset summarization
At FINRA: Internal MCP servers could wrap market surveillance APIs, compliance rule databases, and enforcement case management systems. An AI agent connecting to these servers gets access to the full suite of capabilities through a single protocol.
