Feeding Dynamic Tools to the LLM

MCP and Claude both use JSON Schema for tool input definitions, which means converting between them is a field rename — no transformation logic required. The fields map one-to-one:

MCP FieldClaude Tool Field
namename
descriptiondescription
inputSchemainput_schema
 
def mcp_to_claude_tools(registry):
    """Convert MCP tool definitions to Claude's tool-use format."""
    return [
        {
            "name": name,
            "description": entry["description"],
            "input_schema": entry["schema"]
        }
        for name, entry in registry.items()
    ]
