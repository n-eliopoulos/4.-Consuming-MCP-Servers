Module 2: Connecting to MCP Servers

Resources and Prompts

Tools are the most common MCP capability, but resources and prompts complete the picture.

Resources

Resources are read-only data identified by URI. Use resources/list to discover them and resources/read to fetch content.

# List available resources
resources = client.list_resources()
# [{"uri": "compliance://rules/3110", "name": "FINRA Rule 3110", "mimeType": "text/plain"}, ...]

# Read a specific resource
content = client.read_resource("compliance://rules/3110")
# Returns the full text of the resource

Prompts

Prompts are reusable templates with arguments. Use prompts/list to discover them and prompts/get to retrieve a filled-in template.

# List available prompts
prompts = client.list_prompts()
# [{"name": "analyze-trade-compliance", "arguments": [{"name": "trade_id", "required": true}, ...]}]

# Get a prompt with arguments filled in
prompt = client.get_prompt("analyze-trade-compliance", {"trade_id": "T-2024-0847", "rule": "3110"})
# Returns the complete prompt text with arguments substituted

When to use each

CapabilityUse whenExample
ToolThe model needs to do somethingCheck compliance, run a query
ResourceThe model needs to read somethingFetch a rule, get a report
PromptYou want a structured templateStandard analysis format
